Página web para un evento de boda.

La web incluye algunas animaciones personalizadas hechas con Tailwind y CSS, con efectos 3D y transiciones llamativas.

Todos los diseños, ilustraciones y dibujos también fueron realizados por mí.

Hay un formulario funcional para confirmar asistencia (RSVP) a la boda. La información se envía a Firebase y la pareja puede acceder, editar y eliminar los datos en un panel creado solo para ellos, con acceso mediante login. También pueden crear grupos y organizar a los invitados.

Usé Playwright para probar los formularios de la web.
