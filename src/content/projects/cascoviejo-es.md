Página web personalizada para un restaurante, construida con Astro y React.

Migré su antigua web de WordPress a un servidor nuevo e implementé el nuevo sitio.

Corregí todos los problemas de SEO, mejoré el posicionamiento y el rendimiento, y añadí Google Analytics.

La web también está conectada a Contentful para que el cliente pueda actualizar el menú, las bebidas y los precios.
