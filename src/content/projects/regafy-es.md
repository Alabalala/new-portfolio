Aplicación web de tipo red social para compartir listas de deseos y regalos con tus amigos.

Puedes compartir regalos, añadir amigos, crear eventos, comentarios, preguntas, intercambios de regalos secretos ¡y mucho más!

La app incluye soporte de i18n y modo oscuro.

Usé Supabase para la autenticación, el almacenamiento y la base de datos.

Necesitas registrarte para usar la app, pero también puedes contactarme para una demo en directo o una cuenta de prueba.
